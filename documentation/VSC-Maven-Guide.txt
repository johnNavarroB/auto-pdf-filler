======    First steps: install VS Code, Java and Maven

1. Download VS Code (https://code.visualstudio.com/download)

2. Download Maven (https://maven.apache.org/download.cgi)

3. Follow installation instructions (https://maven.apache.org/install.html)
   |
   L-> Environment variable "path" must be updated with the \bin java directory as new value
       Env. variable "JAVA_HOME" must be created with the "root" java directory as value
